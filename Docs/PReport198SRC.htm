PReport 1.9.8
    New features
    • New TprBarCodeObj object is added. See "30_BarCodes" demo for details.
    • Improved RTF support - now images in the RTF text are supported.
    • Borders can be defined for TprTxMemoObj. Border is defined by the eight chars, see properties of TprTxMemoObjRecVersion: BrdLeft, BrdLeftTop, BrdTop, BrdRightTop, BrdRight, BrdRightBottom, BrdBottom, BrdLeftBottom. In TxRO.ini file can be defined the "border schemas", each scheme defines the set of border's chars. These schemas can be selected in the designer of the TprTxMemoObj. See help on TprTxBorderScheme.
    • Added native support of JPEG images.
    Removed bugs
    • Property DataSet.LineNo does not work in parser. Fixed. (Bug was added in the 1.9.7 version)
    • Minor bugs fixed in the algorithm of objects and bands splitting.
