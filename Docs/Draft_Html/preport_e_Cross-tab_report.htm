This report is intended for printing table with variable number of columns. During report building, all off-bounds columns will be printed on new page (like in MS Excel).

The Cross-tab report consists of two bands: TprHDetailBand (horizontal detail band) and TprVDetailBand (vertical detail band). Object, which derives contents of the table, should lay on intersection of these two bands. The value is showed in this object may be calculated on the base of key fields in datasets linked with horizontal and vertical bands. The example of data scheme for a Cross-tab report:

  • The horizontal dataset HorDataset (to which a TprHDetailBand is linked) has a key field CDY.
  • The vertical dataset VertDataset (to which TprVDetailBand is linked) has a key field CDX.
  • The third Dataset which is contained fields CDX, CDY, S; where S, for example, the sum for a combination of fields CDX and CDY.

Do the following for create a simple Cross-Tab report:

  • Create the new project
  • Put the TTable component on the form and rename it to HorDataset, DatabaseName = "a path to the table1.dbf file", TableName = table1.dbf. This dataset forms rows of table.
  • Put the TTable component on the form and rename it to VertDataset, DatabaseName = "a path to the table2.dbf file", TableName = table2.dbf. This dataset forms columns of table.
  • Put the TprReport component on the form and open the report designer. Add the report title, put the TprMemoObj object with text "Simple CrossTab demo", for example.
  • Add Detail band (horizontal detail band), set dataset property to HorDataset and check CheckBox "Use vertical bands".
  • Add Vertical Detail band, set dataset property to VertDataset and check CheckBox "Use horizontal bands". Attention! If you click on the intersection of vertical and horizontal bands, the properties window of horizontal band will be shown, you must click within the vertical band area to select it.
  • Add Detail header band (horizontal title of detail band), call a properties of it and on page "Data" in a field "Detail band" indicate prHDetailBand1, and then check "Use vertical bands".
  • Add Vertical detail header band (vertical title of datas), call a properties of it and on page "Data" in a field "Detail band" indicate prVDetailBand1, and then check "Use horizontal bands".
  • Put object TprMemoObj with text [VertDataset. Name] on intersection of Detail header and Vertical detail bands, it forms the horizontal title of CrossTab table, in object indicate.
  • Put object TprMemoObj with text [HorDataset. Name] on intersection of Detail and Vertical detail header bands, it the forms the vertical title of CrossTab table.
  • Put object TprMemoObj on intersection of Detail and Vertical detail bands, it forms a contents of CrossTab table, write within object text:
    HorDataset: [HorDataset. LineNo ()]
    VertDataset: [VertDataset. LineNo ()]

The creation of Cross-tab report for TprTxReport happens absolute similarly, except that instead of TprMemoObj is necessary to use TprTxMemoObj. In example "05_SimpleCrossTab" both variants are showed, for TprReport (Windows report) and for TprTxReport (text report).
