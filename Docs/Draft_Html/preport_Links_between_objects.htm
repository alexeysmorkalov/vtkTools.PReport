PReport
 
Objects links

In PReport the mechanism of dynamic change of the sizes and positions of objects is realized. Size and position of object may depend from its contents and from contents of other objects. For example object TprMemoObj (or TprTxMemoObj) can change the sizes depending on the text and the size of a font. Also between objects of the report template the links can be defined which determine dependences between objects. In PReport can be links of the four types:

  • The height of the object changes depending on other objects. These links are established from bottom side of the object.
  • The width of the object changes depending on other objects. These links are established from right side of the object.
  • The X coordinate of object changes depending on other objects. These links are established from left side of the object.
  • The Y coordinate of object changes depending on other objects. These links are established from top side of the object.

About that how define links between objects in the report designer you can to read in section "Designer". To edit the work mode of links use popup window opened by button. In this window you can delete links and edit properties TopMode, LeftMode, HeightMode and WidthMode.


The links from object are specified by these properties:

property TopObjs: TprObjs read;
Contains the list of objects with which links from the top side of object are established (influencing on Y coordinate of object).
property TopMode: TprVLinkMode read write default prlmMaxBottom;
Specifies how the links are influencing on Y coordinate of object.
TprVLinkMode = (prlmMaxBottom, prlmMinBottom);
prlmMaxBottom - coordinate Y will change on so much as far as the bottom coordinate of the lowest object will change.
prlmMinBottom - coordinate Y will change on so much as far as the bottom coordinate of the highest object will change.
property LeftObjs: TprObjs read;
Contains the list of objects with which links from the left side of object are established (influencing on X coordinate of object).
property LeftMode : TprHLinkMode read write default prlmMaxRight;
Specifies how the links are influencing on X coordinate of object.
TprHLinkMode = (prlmMaxRight, prlmMinRight);
prlmMaxRight - coordinate X will change on so much as far as the right coordinate of the lowest object will change.
prlmMinRight - coordinate X will change on so much as far as the right coordinate of the highest object will change.
property HeightObjs: TprObjs read;
Contains the list of objects with which links from the bottom side of object are established (influencing on height of object).
property HeightMode: TprVResizeMode read write default prrmMaxBottom;
Specifies how the links are influencing on height of object.
TprVResizeMode = (prrmMaxBottom, prrmMinBottom, prrmMaxHeight, prrmMinHeight, prrmHeightSum);
prrmMaxBottom – height of the object will change on so much as far as the bottom coordinate of the lowest object will change.
prrmMinBottom – height of the object will change on so much as far as the bottom coordinate of the highest object will change.
prrmMaxHeight – height of object is defined to equal height of the highest object.
prrmMinHeight – height of object is defined to equal height of the lowest object.
prrmHeightSum –height of object changes on size equal to total change of height of objects.
property WidthObjs: TprObjs read;
Contains the list of objects with which links from the right side of object are established (influencing on width of object).
property WidthMode: TprHResizeMode read write default prrmMaxRight;
Specifies how the links are influencing on width of object.
TprHResizeMode = (prrmMaxRight, prrmMinRight, prrmMaxWidth, prrmMinWidth, prrmWidthSum);
prrmMaxRight – width of the object will change on so much as far as the right coordinate of the lowest object will change.
prrmMinRight – width of the object will change on so much as far as the right coordinate of the highest object will change.
prrmMaxWidth – width of object is defined to equal width of the widest object.
prrmMinHeight – width of object is defined to equal width of the narrowest object.
prrmHeightSum – width of object changes on size equal to total change of width of objects.

Bands links

For section links to objects inside section can be established. These links can change size of the band. The band links are specified by the properties:

property ResizeObjs: TprObjs read;
Contains the list of objects with which links are defined.
property ResizeMode: TprBandResizeMode read write default prbrmNone;
Specifies how the links are influencing on size of the band.
TprBandResizeMode = (prbrmNone, prbrmMaxObj, prbrmMaxResizeObj, prbrmMinResizeObj);
prbrmNone – size of the band does not change.
prbrmMaxObj – size of the band defined as the
prbrmMaxResizeObj
prbrmMinResizeObj

About that how define links between objects in the report designer you can to read in section "Designer". To edit the work mode of links use popup window opened by button or band property window. In these windows you can delete links and edit value of the ResizeMode property.
