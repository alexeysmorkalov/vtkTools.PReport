PReport
 

Follow these steps to create the report:

  • Prepare the datasets, on the basis of which the report will be formed. You can use standard datasets derived from TDataset (TTable, TQuery and so on) or components derived from the TprDataset class (TprStringsDataset, TprEventsDataset).
  • Create the report template.
  • Call the PrepareReport method to generate the report.
  • Call the PreviewPreparedReport method to preview report or PrintPreparedReport to print the prepared report.
