PReport
 

The TxRO.ini file contains all options that are described the printing in the text mode. It is a usual INI file. At the program startup it is searched by the algorithm:
  • In the current application's directory.
  • In the Windows system directory (WINDOWS\SYSTEM).
  • In the Windows directory.
  • In the resources of the program. (TXRO - the resource's name).
The files' sections:
[General] section:
DefaultRecodeTable - contains the name of recode table that will be used by default, if it is not specified in the TprTxReport.RecodeTableName property.
Example:
[General]
DefaultRecodeTable=Russian866and1251
[TxCommands] section:
Contains the list of all commands that are supported by TprTxReport, each value of this section describes one command.
Example:
txcReset = @ Reset priter settings to default

txcReset - The unique command's name.
@ - The unique prefix of the command, can consists of one or tree symbols.
Reset printer settings to default - The command's description.
[TxFontStyles] section:
Contains the font styles, that are supported by TprTxReport, each value of this section represents one font style.
Example:
tfsNormal = txcNormal,Normal

tfsNormal - The unique name of the font style.
txcNormal - The name of command from [TxCommands] section, that is used for initializing this style.
Normal - The description of style.
TxFontOptions section:
Contains the font options that are supported by TprTxReport (bold, italic and so on), each value of this section describes one font option.
Example:
tfoCondensed = txcCondensedOn,txcCondensedOff,Condensed

tfoCondensed - The unique name of command.
txcCondensedOn - The command that is switch on the font option.
txcCondensedOff - The command that is switch off the font option.
Condensed - The description
[RecodeTable_XXX] sections:
The file can contain the many sections of this type, each section describes one recode table. XXX - the name of the recode table, this name must be specified in the TprTxReport.RecodeTableName property.
The section's values:
Description - The description of the recode table.
OEMtoWINTable - The array for recoding OEM -> ANSI.
WINtoOEMTable - The array for recoding ANSI -> OEM. This parameter can be omitted in this case it will be generated automatically.

[ESCModel_XXX] sections:
The file can contain the many sections of this type, each section describes one ESC model and is used for converting between TprTxReport commands and the native commands of printer.
The section's values:
ModelName - The name of the model, this name must be specified in the TprTxReport.ESCModelName.
PrinterDriver - The string that is used for searching the ESC model by the Windows printer driver name, it contains a string which must be contained in the printer driver name.
Example:
ModelName=Epson printers
PrinterDriver=epson
