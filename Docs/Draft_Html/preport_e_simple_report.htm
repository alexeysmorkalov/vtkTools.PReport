Directory: PReport\DEMO\Demos\01_SimpleDemo

In this example you will known how to create a simple report. This report prints all records from the CUSTOMERS table from the DBDEMOS alias. Report contains the report title (TprHTitleBand) and the detail (TprHDetailBand).

  • Create new Delphi project. Put TTable component on the form, set it properties to: DatabaseName = DBDEMOS, TableName = CUSTOMER.DB, then put TprReport component. Double-clicks on the icon of TprReport component to edit the report template.
  • Add the report title band - click on a button and select "Report title" from appear menu.
  • Put into the created band TprMemoObj object, click on a button for this. Double-clicks on the memo object and in the appeared window on the page "Text" enter the report title, for example "List of customers". The object has all borders by default, it can be changed on page "Borders".
  • Add detail band (click on button and select "Detail").
  • Open the properties window for the added detail band (double-clicks on detail band or press button for this).
  • Go to "Data" page and select dataset named "Table1" from the list in the appeared window.
  • Do the following for print the dataset fields:
    • Add the object TprMemoObj with text "[Table1.CustNo]" - this memo will print the value of CustNo field.
    • Add the object TprMemoObj with text "[Table1. Company] (Country - [Table1.Country])" - this memo will print the values of two fields: Company and Country.
