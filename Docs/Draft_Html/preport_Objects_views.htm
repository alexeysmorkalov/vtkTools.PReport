PReport
 

In PReport each object can have many views. View is a set of the objects parameters (text, font, background and so on).

Each view has a formula expression that determinates when the view must be used. The view is used if their formula returns true. If all formulas return false the default view is used, this view marked with bold font in the report designer.

In the report designer, the formulas can be edited in the object properties window.
