PReport
 

Under aggregated values are understood the report summary, group summary, group average and so on. For calculation of such values in the PReport the variables are used, these variables are represented by the TprValue class. The variables can be system and user. The system variables are added automatically by report, in current version of PReport two system variables exist:

  • Page - number of the current page, begins with 1.
  • PagesCount - total amount of pages in the generated report.

The user variables created by developer when he designs the report template.
The variables can be used in the view formulas, in content of memo objects. For example to drawn the current page number and pages count place the TprMemoObj object to the report template and write in them: "Page [Page] from [PagesCount]". Order of variables creating is described below.
