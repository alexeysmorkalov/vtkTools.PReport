PReport
 

This report is intended for printing table with variable number of columns. During report building, all off-bounds columns will be printed on new page (like in MS Excel).

For creating this report, place "Detail" and "Vertical detail" bands on the page. Place object in the cross of these bands. This object will be printed as the cell of cross-table, for Detail band set UseVerticalBand property to true and for Vertical detail band set UseHorizontalBands property to true. That's all! If you want, place "Vertical detail header" and "Vertical detail footer" bands in your report. "Vertical detail footer” band is usable if you want to print sum of data in the row.

Also you can use “Vertical group footer” band and others.
