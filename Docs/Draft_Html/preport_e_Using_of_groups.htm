We create the report, in which the records are grouped by Country field, and then by the first letter of a Company field on the base of report from 01_SimpleDemo. The TQuery component will be used as a data source. Set the TQuery.SQL property to: "select CustNo, Country, Company from customer order by country, company". Don't forget to set a Dataset property of prHDetailBand1 object to Query1.

The groups' editor is called through item "Groups…" of the menu "Edit". Use button for creating a new group. You must specify for first group:

  • Group expression - the condition will be Query1.Country for the first group.
  • Detail band to which the group is linked, in this case - prHDetailBand1.

For the second group:

  • Group condition will be: UID(Query1.Country, Copy(Query1.Company,1,1)). As you see an expression of the enclosed group uses an expression of the parent group, it is important!
  • Detail band - prHDetailBand1.

Create a group title for each group. The group title is located after all bands on page after adding; it rise on the necessary place when you select a group for added band. The group can be indicated in a field Group on a page "Data" of the properties window.
