You will know how to create a multicolumn report in this example. This report prints all records from the CUSTOMERS table of the DBDEMOS alias in two columns.

  • Create new Delphi project. Put TTable component on the form, set its properties to: DatabaseName = DBDEMOS, TableName = CUSTOMER.DB, then put TprReport component.
  • Double-clicks on the icon of TprReport component to edit the report template.
  • Add the report title band, click on a button for this and select "Report title" from the appeared menu.
  • Put TprMemoObj object into the created band, click on a button for this. Double-clicks on the memo object and in the appeared window on the page "Text" enter the report title, for example "List of customers". The object has all borders by default, it can be changed on page "Borders".
  • Add detail band (click on button and select "Detail").
  • Open the properties window for the added detail band.
  • Go to "Data" page and select dataset named "Table1" from the list in appeared window.
  • Go to "Columns" page and enter 2 in field "Columns count", (report will have two columns).
  • Do the following for print the dataset fields:
    • Add the object TprMemoObj with text "[Table1.CustNo]" - this memo will print the value of CustNo field.
    • Add the object TprMemoObj with text "[Table1. Company] (Country - [Table1.Country])" - this memo will print the values of two fields: Company and Country.

In this demo also is demonstrates how to change the mode of creation multicolumn report. Value of the TprHDetailBand.ColDirection property is changed in the event handler of Button1 button, this value depends from the user selection. The TprHDetailBand.ColDirection property can have one of two values:
prcdTopBottomLeftRight - the columns is created from top to down then from left to right
prcdLeftRightTopBottom - the columns is created from left to right then from top to down
