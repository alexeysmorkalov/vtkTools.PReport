Object toolbar
Cut selected objects to clipboard (bands and pages can not be copied, only objects within bands).
Copy selected objects to clipboard (bands and pages can not be copied, only objects within bands).
Paste object from clipboard into the currently selected band.

Shows a popup window that allows editing links from the currently selected object.

Object can has links of the four types:

  • From left side, these links change X coordinate of object.
  • From top side, these links change Y coordinate of object.
  • From right side, these links change width of object.
  • From bottom side, these links change height of object.

To define link from object do the following:

  • Move mouse to the needed side of the object.
  • Press left mouse button and move mouse to the other object (to which link must be defined)
  • Leave mouse button.
  • Link must be created and shows as black line from one object to other.
Please see demo “14_NewObjectsLinksModes”.
