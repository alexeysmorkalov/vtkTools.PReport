Object prMemoObj1 (Table1. Notes)

Page in the properties window Field Value Comment
Addititional Resize vertically true The size of object can be dynamically changed depending on size of the text in the field Table1.Notes
Word wrap true  

Object prMemoObj6 (Length: [Table1.Length_In])

This object should be under the object prMemoObj1. The height of the prMemoObj1 object can be changed dynamically therefore for object prMemoObj6 the connection to object prMemoObj1 is installed. This connection can be viewed in the window of the object connections (can be called by a button ) on page "Top"; the field mode is installed to "Y coord depends from max bottom" on this page. Except for this object the field "Resize horizontally" is installed, it width will be determined by a size of the text in it.

Object prImageObj1 (Table1.Graphic)

The size of this object is determined by a size of an image in it, the "Paint method" field is installed to "Sizes depends from image size".

Objects prMemoObj4 (Common name) and prMemoObj2 (Table1.Common_Name)

These objects should be under prImageObj1. The height of prImageObj1 can be changed dynamically, therefore for objects prMemoObj4 and prMemoObj2 the connection to object prImageObj1 is installed.

Objects prMemoObj5 (Category) and prMemoObj3 (Table1.Category)

These objects should be under objects prMemoObj4 and prMemoObj2. The coordinate Top of the prMemoObj4 and prMemoObj2 objects can be changed dynamically therefore for objects prMemoObj5 and prMemoObj3 the connection accordingly to objects prMemoObj4 and prMemoObj2 is installed.

For the detail band (prHDetailBand1) the field "Size" is installed to "Size depends from max bottom in objects".
